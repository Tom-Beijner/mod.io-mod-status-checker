# Mod.io mod status checker

A JavaScript based application to check if new mods become hidden (or deleted) and sends a webhook message about the newly affected mod.

## Installation

Use the package manager npm or yarn

```bash
npm install
```

or

```bash
yarn install
```

## Usage

```bash
npm run start
```

or

```bash
yarn start
```
